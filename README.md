## 🛠 Standard Operating Procedures (SOPs)

* **The "Clock" Method:** 12 (Well Done), 3 (Medium), 6 (Med-Rare).
* **Resting Rule:** Minimum 5-7 minutes for all steaks under foil.
* **Cleaning:** Scrape, water-steam, and light oil coat after every session.
